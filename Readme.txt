# YouTube Video & Audio Downloader

A simple and fast YouTube video and audio downloader built using Streamlit and Pytubefix. It allows users to download YouTube videos in different resolutions and extract audio as MP3.

## Features
- Download high-quality YouTube videos in multiple resolutions.
- Download audio-only streams as MP3 files.
- Real-time progress bar displaying the download and merging status.
- Clean, user-friendly UI with responsive design.
  
## Technologies Used
- **Streamlit**: For building the web interface.
- **Pytubefix**: For downloading YouTube video/audio streams.
- **FFmpeg**: For merging video and audio streams.
  
## Setup & Installation

### Prerequisites

- Python 3.6 or higher
- FFmpeg installed on your system. You can install FFmpeg via [FFmpeg website](https://ffmpeg.org/download.html) or through package managers.
