
YouTube Video Downloader (with Audio Merging)
This script allows you to download YouTube videos in various resolutions and also provides an option to download the audio as an MP3 file. The script supports downloading high-quality videos and combining video with audio into a single MP4 file.

Features:
Download high-quality videos in MP4 format.
Download audio-only in MP3 format.
Choose the video quality (resolution).
Merge video and audio using ffmpeg.


To run the app
- Go to your your directory for app
- run this command in your terminal  "streamlit run file_name.py"
